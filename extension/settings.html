Control Options
Keyboard navigation is as follows:
  • N - Next Post
  • P/M - Previous Post
  • J - Next Page
  • K/H - Previous page
  • 1/F - Jump to first post on page
  • 0/L - Jump to last post on page
  • O - Reanchor thread
  • Q - Quick Quote current post
  • E - Quick Edit current post
  • R - Quick Reply current thread
  • ESC - Close Quick Reply/Edit box
  • B - (When in UCP/Showing Forum/Bookmarked Threads) Hit twice to open all bookmarked threads
Note: This checkbox is checked by default when editing bookmarked threads. Unselecting this option on editing posts will unbookmark this thread. Unselecting this option when just regularly posting won't unbookmark the thread.
When you paste an image or youtube video link into the quick reply, it will be automatically parsed into the image/video tags for you.
